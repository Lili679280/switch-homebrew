# Switch Button Icon Font

`NintendoSwitchButtonIcons.ttf` is a renamed, byte-identical copy of the font identified as `NintendoExt003` by the download source. The internal font-family name remains `NintendoExt003`.

The glyphs use Unicode private-use codepoints. Consult `../../references/button-icon-font.md` before integrating the font. Do not guess which codepoint represents a controller button.

The font is included for non-commercial Nintendo Switch homebrew development. See `LICENSE.txt` and the repository's `THIRD_PARTY_NOTICES.md` before redistributing it.
