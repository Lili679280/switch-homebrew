# Security Policy

This repository contains development guidance and bundled upstream references; it is not an official Nintendo SDK.

When reporting a security issue:

1. Do not publish active credentials, private keys, personal data, or working exploit details in a public issue.
2. Prefer GitHub's private vulnerability reporting feature for this repository when available.
3. Include the affected file, bundled component/version, reproduction conditions, and expected impact.
4. For vulnerabilities in libnx or switch-examples themselves, also report the issue to the appropriate upstream project.

Generated homebrew code must still be reviewed, compiled, and tested by the developer. This skill does not guarantee that generated code is safe for every firmware, environment, or privilege level.
