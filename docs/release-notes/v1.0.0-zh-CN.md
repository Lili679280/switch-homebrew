# Switch Homebrew Skill v1.0.0

> 面向 Nintendo Switch 自制程序开发的自包含 AI Skill 首个稳定版本。

[English →](https://github.com/Lili679280/switch-homebrew/blob/main/docs/release-notes/v1.0.0-en.md)

---

## 概览

这个版本提供一套可以直接使用的 Switch 自制程序开发参考资料。它把 libnx 头文件、官方示例、导航索引和 Switch 按钮图标字体放在同一个 Skill 中，让 AI 编程助手能够从内置资料核对 API，而不是猜测函数签名或依赖某台电脑上的固定源码路径。

## 内置内容

- **libnx v4.12.0**：内置官方签名标签的源码与头文件
- **switch-examples**：内置官方示例，固定提交为 `669786898205b7beb25ff1731e72982e6d0397d3`
- **API 索引**：根据开发子系统定位相应的权威头文件
- **示例索引**：定位内置官方示例中的实际用法
- **按钮图标字体**：Nintendo Switch 按钮 TTF 与渲染说明
- **项目文档**：相互独立的英文和简体中文 README
- **仓库规范**：贡献说明、安全策略、Issue 与 Pull Request 模板

## 安装

将仓库克隆到个人 Codex Skills 目录：

```powershell
git clone https://github.com/Lili679280/switch-homebrew.git "$env:USERPROFILE\.codex\skills\switch-homebrew"
```

安装后重新启动 Codex，或者新建一个任务，让 Codex 重新发现这个 Skill。

## 下载

GitHub 会在 [v1.0.0 Release 页面](https://github.com/Lili679280/switch-homebrew/releases/tag/v1.0.0)自动提供完整项目的 **Source code (zip)** 和 **Source code (tar.gz)** 压缩包。

内置的第三方内容继续遵循各自的许可条款，详情请查看 [THIRD_PARTY_NOTICES.md](https://github.com/Lili679280/switch-homebrew/blob/main/THIRD_PARTY_NOTICES.md)。
