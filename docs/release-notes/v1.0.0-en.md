# Switch Homebrew Skill v1.0.0

> First stable release of the self-contained AI skill for Nintendo Switch homebrew development.

[中文版 →](https://github.com/Lili679280/switch-homebrew/releases/tag/v1.0.0)

## Overview

This release provides a ready-to-use, self-contained reference set for AI-assisted Nintendo Switch homebrew development. It keeps libnx headers, official examples, navigation indexes, and the Switch button icon font together so coding agents can verify APIs locally instead of guessing signatures or relying on machine-specific source paths.

## Included

- **libnx v4.12.0** — bundled official signed-tag sources and headers
- **switch-examples** — official examples pinned at commit `669786898205b7beb25ff1731e72982e6d0397d3`
- **API index** — locates the authoritative bundled header for each subsystem
- **Examples index** — locates official usage patterns in the bundled examples
- **Button icon font** — Nintendo Switch button TTF and rendering guidance
- **Project documentation** — separate English and Simplified Chinese README files
- **Repository standards** — contribution, security, Issue, and Pull Request templates

## Installation

Clone the repository into your personal Codex skills directory:

```powershell
git clone https://github.com/Lili679280/switch-homebrew.git "$env:USERPROFILE\.codex\skills\switch-homebrew"
```

Restart Codex or start a new task after installation so the skill can be discovered.
